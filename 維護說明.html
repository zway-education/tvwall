0. 兩個檔案 ・ 一個工具

檔案是什麼何時開
index.html電視牆本體(畫面+程式都在這)想改內容時
assets/portrait.jpg(可選)創辦人肖像想換照片時

改內容的工具:用 記事本 Notepad 就夠了(右鍵 index.html → 開啟方式 → 記事本)。
推薦 VS Code 或 Notepad++,中文比較好看。

1. 「編輯區」在哪裡?

打開 index.html,按 Ctrl + F 搜尋「編輯區」,游標會跳到一個被框框包起來的區段:

╔════════════════════════════════════════════════════════════════╗
║      👇  編 輯 區  ・  改 內 容 只 動 這 一 段 就 好  👇         ║
╚════════════════════════════════════════════════════════════════╝
⚠ 只要在這個框框內改,框框外面的程式都不要動。 改完按 Ctrl + S 存檔。

2. 改完後怎麼看到結果?

  1. 把電視牆瀏覽器調出來
  2. 按 F5 重新整理
  3. 看見新內容 ✅

如果是電視主機在櫃檯背後,可以用滑鼠/遙控器移到網頁區按 F5,或重啟瀏覽器。

3. 最常改的 4 件事

3.1 換 LINE 加好友連結

位置:編輯區 → ❶ QR 連結

qr: {
  line: "https://lin.ee/nnDYAZE",      // ← 改這個引號內的網址
  ...
}
LINE 加好友連結請從 LINE 官方帳號管理後台複製,確認是 https://lin.ee/xxxxx 格式。

3.2 換心智光譜連結

同一個位置:

qr: {
  ...
  mindspectrum: "https://zway-education.github.io/mindspectrum-advanced/",
}

3.3 換 / 加 / 減 真實見證

位置:編輯區 → ❷ Slide 4 真實見證

每條見證長這樣(注意每條最後有逗號 ,):

{
  text: '我兒子以前回家就摔門。<br>現在他會說<b>「媽媽,我今天心情有點亂。」</b>',
  who:  '— 國中智優 ・ 陳媽媽(高雄)'
},

寫作小技巧:

  • 用 <br> 換行
  • 用 <b>...</b> 框出要強調的關鍵字(會自動加金色底線)
  • 引號用單引號 '...',見證內若要用引號請用中文引號 「」 或 “”
  • 不要用程式語法的雙引號 " 在見證內容裡(會把字串斷掉)
  • 化名規則:「陳媽媽 / 林爸爸 / 楊老師」+ 班別 + 城市,不寫真實姓名

加新見證:複製整個 { ... } 區塊,貼在後面,逗號別漏。
刪見證:整個 { ... }, 連逗號一起刪。
最少 1 條,最多沒上限(但 5 條以上輪播太久,觀眾看不完)。

3.4 改 slide 停留秒數

位置:編輯區 → ❸ 每張 slide 停留秒數

durations: {
  s1: 15000,  // ← 15000 毫秒 = 15 秒
  s2: 12000,
  ...
}

參考值:

  • 太快(< 8 秒):觀眾來不及讀完
  • 剛好(10-15 秒):大多數人能讀完並開始思考
  • 太慢(> 20 秒):走過的家長等不及切下一張就走了

4. 偶爾才改的 3 件事

4.1 改創辦人肖像

  1. 找一張守蕾老師正式照(直幅 3:4 比例,建議 1200×1600 以上)
  2. 命名為 portrait.jpg
  3. 放進 assets/ 資料夾(沒有就自己建一個)
  4. 改 index.html 內 Slide 3 區塊:
    • 搜尋「守蕾老師<br>肖像區」
    • 把整個 <div class="ph-text">...</div> 換成
      <img src="assets/portrait.jpg" style="width:100%;height:100%;object-fit:cover">

4.2 改 Slide 主視覺文字

不建議改,「先懂心,再懂教」是品牌核心句。

真的要改:搜尋「先懂心」,找到 <h1 class="headline">...</h1>,改引號之間的字。字太多會跑版,試播一輪看一下。

4.3 改 4 個班別名稱

不建議改。班別名稱是品牌資產。

真的要改:搜尋「啟蒙」或「開智」,找到 Slide 2 的 4 個 <div class="stage">...</div>,改裡面的 age / name / tag 文字。

5. 萬一搞砸了

救命方法 1 ・ 復原

記事本 / VS Code 按 Ctrl + Z 一直按到回到原狀。還沒存檔就還救得回來。

救命方法 2 ・ 從備份還原

建議改之前先複製一份備份:右鍵 index.html → 複製 → 貼上,系統會自動命名 index - 複製.html。改壞了就把備份改名回 index.html。

救命方法 3 ・ 找 Claude

跟我說「電視牆壞了,顯示空白 / 切不過去 / XX 不見了」,我會幫你看。

常見壞掉症狀對照表

症狀原因怎麼修
電視牆完全空白編輯區有引號 ' 沒成對用備份還原,或檢查每對引號
QR 碼掃不到URL 拼錯、缺 https://補上 https:// 並確認 URL 正確
見證跳得太快/慢testimonyInterval 數字改錯改回 4000(4 秒)
字跑出畫面改的文字太長縮短,或加 <br> 換行
中文變亂碼存檔編碼不對(Notepad 預設 ANSI)另存新檔 → 編碼選 UTF-8

6. 進階:多台部署

換到另一台電腦

  1. 整個 電視牆/ 資料夾複製到新電腦任意位置
  2. Chrome 開 index.html,F11 全螢幕,結束。

多個分校用同一套

把整個資料夾複製過去,在每台分校的 index.html 編輯區改成在地化內容(例如見證換成該分校的)。

想自動開機跑全螢幕

詳細的 Chrome kiosk 模式設定見 docs/superpowers/specs/2026-05-13-counter-tvwall-design.md 的 §5.3 章節。
