常見問題

Q. 守蕾老師也想能改怎麼辦?

她在手機 Safari/Chrome 開同一個 URL,加到主畫面就好。API Key 是共用的,所有有 URL + Key 的裝置都能改。

Q. 怕被亂改怎麼辦?

  1. Sheet 編輯歷史會留所有變更(右上角:檔案 → 版本記錄)
  2. 誰改的會記在 log 工作表(看 updated_by 欄位)
  3. 想加密碼?跟我說,我可以加一個簡單密碼鎖

Q. Web App URL 不小心外流怎麼辦?

到 Apps Script 編輯器 → 部署 → 管理現有部署作業 → 編輯 → 建立新版本 → 部署。會給新 URL,把舊的失效。同時改 cloud_config.js 內 API_KEY 換成新字串(同步改 Apps Script 內 API_KEY 常數)。

Q. 雲端壞了怎辦?

本機 cache 還在,電視牆會繼續播最後一次抓到的內容。修好後自動恢復。
